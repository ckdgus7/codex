# Product

NOVA AI DevOps is an enterprise-grade project management and development lifecycle platform. It provides tools for requirements management, UI design tracking, feature/function management, SSF (System Structure Framework) hierarchy management (Domain L1 → Component L2 → Business L3 → Function L4), SBF (Standard Business Framework) lifecycle management, and team collaboration via notices and Q&A boards.

The platform targets development teams managing complex software projects, with a vertical sidebar navigation layout and MDI (Multi-Document Interface) tab system for multi-page workflows.
