# Tech Stack

## Projects

## preview_react (Primary App)

- **React 19** with TypeScript 5.9
- **Vite 7** — build tool and dev server (port 5009)
- **React Router 7** — client-side routing
- **TanStack React Query 5** — server state management
- **Zustand 4** — client/UI state management
- **bpmn-js** — BPMN 2.0 diagram rendering
- **@toast-ui/editor** — WYSIWYG rich text editor
- **Storybook 10** — component development and documentation (port 6006)
- **Path alias**: `@` → `./src`

## Common Commands

### preview_react
```bash
cd preview_react
npm run dev          # Dev server on port 5009
npm run build        # tsc -b && vite build
npm run preview      # Preview production build
npm run storybook    # Storybook on port 6006
npm run build-storybook
```
